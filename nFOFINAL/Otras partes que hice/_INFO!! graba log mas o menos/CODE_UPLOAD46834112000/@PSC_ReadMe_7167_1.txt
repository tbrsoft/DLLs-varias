Title: WIS
Description: This code is a windows system information. You can retrive your system information in an *.INI file. The informations you can retrieve are: OS version(Win version, licence number, licence owner, licence company, time zone) - Sys Report(bios, processor) - Video (bios, chip, size,..) - the SET variables - NetWork (hostname, ip, mask, dns, dhcp,domain,...) - Keybord Type - IE version - Disk(type, filesystem, space, freespace, ...) - Services started, ... - Services Stoped, ... - Input host file - Input LmHost file.
